# Archival Freeze

`qrwkv-xla` is frozen as the historical QRWKV-XLA / RADJAX monorepo.

The monorepo reached the pipeline verification / validation point needed to
begin the split into working repositories. Active development now continues in:

- [RADJAX-Contract](https://github.com/nova-rey/RADJAX-Contract) — Tome contract, validation, cover page, and compression metadata
- [RADJAX-Tome](https://github.com/nova-rey/RADJAX-Tome) — teacher-side Tome / TeacherTextbook generation
- [RADJAX-Student](https://github.com/nova-rey/RADJAX-Student) — student-side Tome consumption, training, evaluation, and export

This repository is retained as historical source material, evidence trail, and
migration reference. New implementation work should not be committed here unless
the repository is explicitly unarchived for archival repair.

Historical phase docs, roadmap entries, and experiment notes below this freeze
point may still describe recommended next steps from the monorepo era. Treat
those as historical records, not as instructions to continue implementation in
this repository.
