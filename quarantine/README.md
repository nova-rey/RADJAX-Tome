# Quarantine

This directory stores non-importable archived `qrwkv-xla` reference material for
RADJAX-Tome migration work.

Files under `quarantine/qrwkv_xla/` are copied as `.txt` with headers explaining
their original path and quarantine reason. They are not runtime code, not public
API, and not imported by `radjax_tome`.

See `docs/TOME_GENERATOR_QUARANTINE.md` and
`docs/TOME_GENERATOR_BULK_MIGRATION_MANIFEST.json` for the path-level manifest.
